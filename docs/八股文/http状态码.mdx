- 10X (信息响应)
- 20X (成功响应)
  - 200 OK 请求成功
  - 201 Created 请求成功,并且创建了一个新的资源
  - 202 Accepted 请求已经收到但是还未响应
- 30X (重定向响应)
  - 300 请求拥有多个可能的响应。用户代理或者用户应当从中选择一个。
  - 301 请求资源的 URL 已永久更改。
  - 304 这是用于缓存的目的。(协商缓存)
- 40X (客户端错误响应)
  - 400 由于被认为是客户端错误(例如，错误的请求语法)，服务器无法或不会处理请求。
  - 401 必须对自身进行身份验证
  - 403 客户端没有访问内容的权限
  - 404 服务器找不到请求的资源。
  - 405 服务器知道请求方法，但目标资源不支持该方法。
- 50X (服务端错误响应)
  - 500 服务器遇到了不知道如何处理的情况。
  - 501 服务器不支持请求方法
  - 502 表明服务器作为网关需要得到一个处理这个请求的响应
